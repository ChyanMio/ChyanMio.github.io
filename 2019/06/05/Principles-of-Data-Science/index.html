Principle Compionent Analysis(PCA)

  1. 两种解释:
    (1) Maximum variance direction: when X is decentralized.

    Lagrangian form: (2) Minimum reconstruction error:

  2. PCA的算法步骤
    设有m条n维数据。

  • 将原始数据按列组成n行m列矩阵X。
  • 将X的每一行（代表一个属性字段）进行零均值化，即减去这一行的均值。
  • 求出协方差矩阵1/mXX^T。
  • 求出协方差矩阵的特征值及对应的特征向量。
  • 将特征向量按对应特征值大小从上到下按行排列成矩阵，取前k行组成矩阵P。
  • Y=PX即为降维到k维后的数据。
  1. 能力和限制
    PCA本质上是将方差最大的方向作为主要特征，并且在各个正交方向上将数据“离相关”，也就是让它们在不同正交方向上没有相关性。因此，PCA可以很好的解除线性相关，但是无法处理非线性相关。

Stochastic Neighborhood Embedding (SNE)

把原空间的数据点之间的距离转换为高斯分布概率，如果两点在高维空间距离越近，那么这个概率值越大。

Local Linear Embedding (LLE)

LLE属于流形学习(Manifold Learning)的一种，关注于降维时保持样本局部的线性特征。

Sparse coding

将输入的样本集X分解为多个基元的线性组合，然后这些基前面的系数表示的是输入样本的特征。

Multi Dimensional Scaling (MDS)

多维尺度变换算法解决的问题是:当n个对象之间的相似性给定，确定这些对象在低维空间中的表示，并使其尽可能与原先的相似性大致匹配。将高维坐标中的点投影到低维空间中，保持点彼此之间的相似性尽可能不变。

ISOMap

  1. Constructing K-nearest neighbour graph G
  2. For each pair of points in G, Computing shortest path: geodesic distance
  3. Use Classical MDS with geodesic distance matrix
